# AliFake - Mobile E-Commerce 🛒

Browse products, compare prices and features, and make dummy purchases with this e-commerce platform inspired by Aliexpress.

It operates as a marketplace where are avaliable products with their own descriptions, and images, including electronics, clothing, and accessories, among others.

Here you can search for any item in the store, save your favorite product into a wish list, add them at your purchase cart and even look at your history of items seen in case you forgot one you loved before.

This project was made thinking specially for mobile devices.

#

### Building and running on localhost

First install dependencies:

```sh
npm install
```

To create a production build:

```sh
npm run build
```

To create a development build:

```sh
npm run build-dev
```

### Running

Open the file `dist/index.html` in your browser
